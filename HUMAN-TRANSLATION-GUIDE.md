# 👨‍💼 人工客服翻译功能使用指南

## 📋 功能概述

当AI翻译不能满足需求时,用户可以请求专业客服提供人工翻译服务。这个功能适合:
- 专业术语翻译
- 文学作品翻译
- 重要商务文档
- AI翻译结果不满意的场景

---

## 🎯 用户使用流程

### 步骤1: 提交翻译请求

1. 登录系统
2. 点击顶部导航 "👨‍💼 人工翻译"
3. 点击 "+ 新建请求" 按钮
4. 填写表单:
   - **日文原文**: 需要翻译的内容(必填)
   - **上下文说明**: 提供背景信息,帮助获得更准确翻译(可选)
   - **优先级**: 
     - 普通: 1-2个工作日内处理
     - 紧急: 6-12小时内处理
5. 点击 "提交请求"

### 步骤2: 等待客服处理

- 请求提交后,状态为 "⏳ 待处理"
- 客服接手后,状态变为 "🔄 处理中"
- 您可以随时查看请求状态

### 步骤3: 查看翻译结果

- 客服完成后,状态变为 "✅ 已完成"
- 翻译结果会显示在请求详情中
- 您会收到邮件通知(如果启用)

### 步骤4: 评价翻译质量

- 查看翻译结果后,可以进行评分(1-5星)
- 可以留下反馈意见
- 帮助我们改进服务质量

---

## 🔑 管理员/客服使用流程

### 步骤1: 查看翻译队列

1. 登录管理员账号
2. 点击顶部导航 "📋 翻译队列"
3. 看到所有待处理的请求

**队列视图包含:**
- 待处理数量
- 处理中数量
- 已完成数量
- 各请求的详细信息

### 步骤2: 处理翻译请求

1. 在待处理列表中选择一个请求
2. 查看:
   - 日文原文
   - 用户提供的上下文
   - 优先级(紧急请求有🔥标记)
3. 点击 "开始处理"
4. 在文本框中输入中文翻译
5. 点击 "✅ 提交翻译"

### 步骤3: 监控服务质量

- 查看用户评分和反馈
- 分析常见问题
- 持续改进翻译质量

## ⚙️ 配置说明

### 数据库设置(部署时一次性配置)

1. 登录Supabase控制台
2. 打开SQL Editor
3. 执行 `human-translation-setup.sql` 文件中的SQL代码
4. 验证表创建成功:
   - 进入 Table Editor
   - 看到 `translation_requests` 表

### 邮件通知设置(可选)

**启用邮件提醒:**
1. Supabase → Authentication → Email Templates
2. 自定义 "Translation Request" 模板
3. 设置SMTP服务器

**建议邮件模板:**

```
主题: 您的翻译请求已完成

尊敬的用户,

您提交的翻译请求已经完成:

原文: {{japanese_text}}
译文: {{human_translation}}

登录查看详情: https://your-site.vercel.app

感谢使用我们的服务!
```

## 🔧 技术细节

### 数据库结构

**translation_requests 表:**
```sql
- id: 请求ID
- user_id: 用户ID
- japanese_text: 日文原文
- context: 上下文说明
- priority: 优先级(normal/urgent)
- status: 状态(pending/processing/completed/cancelled)
- human_translation: 人工翻译结果
- translator_id: 处理客服ID
- rating: 用户评分(1-5)
- feedback: 用户反馈
- created_at: 创建时间
- updated_at: 更新时间
- completed_at: 完成时间
```

### 状态流转

```
pending(待处理)
    ↓
processing(处理中) ← 客服接手
    ↓
completed(已完成) ← 提交翻译
    ↓
(用户评分)
```

### 实时更新

系统使用Supabase的实时订阅功能:
- 新请求自动出现在队列
- 状态变更实时显示
- 无需手动刷新

## 🆘 常见问题

### Q1: 人工翻译需要多久?

**A:** 
- 紧急请求: 6-12小时
- 普通请求: 1-2个工作日
- 具体时间取决于请求复杂度和队列长度

### Q2: 可以修改已提交的请求吗?

**A:** 
- 待处理状态: 可以取消重新提交
- 处理中/已完成: 无法修改
- 建议提交前仔细检查

### Q3: 人工翻译是否收费?

**A:** 
- 取决于你的定价策略
- 建议: 免费用户每月5次,付费无限
- 可以设置配额系统

### Q4: 如何确保翻译质量?

**A:** 
- 客服需通过资质认证
- 用户评分系统
- 定期质量审核
- 低分翻译重新处理

### Q5: 可以指定客服吗?

**A:** 
- 当前版本自动分配
- 企业版可以指定专属客服
- 后续版本可添加客服选择功能

### Q6: 紧急请求加急费用?

**A:** 
- 建议收取加急费
- 例如: 普通$5/次,紧急$10/次
- 或专业版用户免费紧急
