### 🚀 如何使用新功能

#### 快速开始(5分钟)

**第一步: 获取API密钥**

**选项A: Gemini(推荐,免费)**
1. 访问 https://makersuite.google.com/app/apikey
2. 登录Google账号
3. 点击"Create API Key"
4. 复制密钥

**选项B: DeepL(专业,付费)**
1. 访问 https://www.deepl.com/pro-api
2. 注册账号并绑定信用卡
3. 获取API密钥

**第二步: 配置系统**
1. 登录翻译系统
2. 点击"🤖 AI翻译设置"
3. 选择服务(Gemini或DeepL)
4. 粘贴API密钥
5. 完成!

**第三步: 开始翻译**
1. 输入日文文本
2. 查看历史匹配(如果有)
3. 没有匹配?点击"🤖 AI翻译"
4. 查看结果,保存或复制
