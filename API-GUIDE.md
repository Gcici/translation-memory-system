## 1️⃣ Google Gemini配置(免费推荐)

### 为什么选择Gemini?
- ✅ **完全免费** - 每分钟60次请求
- ✅ **质量良好** - Google最新AI技术
- ✅ **简单易用** - 5分钟即可配置
- ✅ **无需信用卡** - 只需Google账号

### 获取Gemini API密钥

**步骤1: 访问Google AI Studio**
1. 打开 https://makersuite.google.com/app/apikey
2. 使用你的Google账号登录

**步骤2: 创建API密钥**
1. 点击 "Create API Key" (创建API密钥)
2. 选择一个Google Cloud项目(如果没有会自动创建)
3. 点击 "Create API key in new project"
4. 复制生成的API密钥(格式: `AIzaSy...`)

**步骤3: 在系统中配置**
1. 登录翻译系统
2. 点击 "🤖 AI翻译设置"
3. 选择 "Google Gemini"
4. 粘贴你的API密钥
5. 完成!

### Gemini免费额度
- **每分钟**: 60次请求
- **每天**: 1,500次请求
- **完全免费**,无需绑定信用卡

**适用场景:**
- 个人使用
- 小团队(< 10人)
- 日常翻译需求

## 2️⃣ DeepL配置(付费高质量)

### 为什么选择DeepL?
- ✅ **翻译质量最高** - 业界公认最准确
- ✅ **专业术语** - 擅长技术和专业文档
- ✅ **自然流畅** - 译文更接近母语表达
- ❌ **需要付费** - 按字符收费

### 获取DeepL API密钥

**步骤1: 注册DeepL API**
1. 访问 https://www.deepl.com/pro-api
2. 点击 "Sign up for free" (免费注册)
3. 填写邮箱和个人信息
4. **需要绑定信用卡**(即使是免费版)

**步骤2: 获取API密钥**
1. 登录DeepL账户
2. 进入 "Account" → "API Keys"
3. 复制你的 "Authentication Key"
4. 格式示例: `abc12345-6789-...`

**步骤3: 在系统中配置**
1. 登录翻译系统
2. 点击 "🤖 AI翻译设置"
3. 选择 "DeepL"
4. 粘贴你的API密钥
5. 完成!

### DeepL费用

**免费版(DeepL API Free):**
- 每月: 500,000字符(约25万汉字)
- 需要绑定信用卡
- 超出后自动停止服务

**付费版(DeepL API Pro):**
- 起步价: $5.49/月
- 包含: 5,000,000字符
- 超出: $25/百万字符
- 按实际使用计费

**费用估算:**
- 1条翻译(平均50字): ~100字符
- 500,000字符 ≈ 5,000条翻译/月

**适用场景:**
- 专业翻译需求
- 大量文档翻译
- 对质量要求高的场合
- 企业使用

## 3️⃣ 使用说明

### 基本使用流程

1. **配置API密钥**
   - 在系统中设置Gemini或DeepL的API密钥
   - 密钥仅保存在本地浏览器,不会上传服务器

2. **进行翻译**
   - 在搜索框输入日文
   - 系统优先显示历史匹配(70%相似度)
   - 如需AI翻译,点击 "🤖 AI翻译" 按钮

3. **处理翻译结果**
   - 查看AI翻译结果
   - 可以选择:
     - ✅ 保存到记忆库(下次可以匹配)
     - 📋 复制翻译文本
     - ✕ 关闭结果

### 智能工作流程

```
输入日文
    ↓
系统搜索历史记录
    ↓
找到匹配? ━━━ 是 ━━━→ 显示历史翻译(免费,最快)
    ↓
    否
    ↓
点击"AI翻译"
    ↓
使用Gemini/DeepL翻译
    ↓
查看结果 → 保存到记忆库
    ↓
下次搜索时优先显示
```

### 最佳实践

**混合使用策略:**
1. **日常使用**: Gemini免费版
2. **重要文档**: 切换到DeepL
3. **常用内容**: 保存到记忆库,下次免费匹配

**节省成本技巧:**
- ✅ 翻译后记得保存到记忆库
- ✅ 充分利用模糊匹配(免费)
- ✅ 批量翻译时先用Gemini初译,重要部分用DeepL
- ✅ 定期导出翻译记录备份

## 5️⃣ 常见问题

### Q1: API密钥安全吗?
**A:** 
- ✅ 密钥仅保存在你的浏览器本地存储
- ✅ 不会上传到我们的服务器
- ✅ 其他用户无法看到你的密钥
- ⚠️ 建议定期更换密钥
- ⚠️ 不要在公共电脑上使用

### Q2: 为什么推荐Gemini?
**A:**
- 完全免费,适合大多数用户
- Google AI技术,质量可靠
- 配置简单,无需信用卡
- 免费额度(1500次/天)足够日常使用

### Q3: DeepL和Gemini翻译质量差别大吗?
**A:**
- 日常对话: 差别不大
- 专业术语: DeepL更准确
- 文学作品: DeepL更自然
- 技术文档: DeepL更可靠
- **建议**: 先用Gemini,重要内容用DeepL

### Q4: 我可以同时配置两个API吗?
**A:** 
- ✅ 可以!系统支持配置两个API
- 你可以随时切换使用哪个服务
- 灵活选择最适合当前任务的API

### Q5: API密钥会过期吗?
**A:**
- Gemini: 不会过期(除非你手动删除)
- DeepL: 不会过期(除非账户被禁用)
- 建议每半年检查一次密钥是否有效

### Q6: 如果超出免费额度怎么办?
**A:**
- **Gemini**: 等待配额恢复(每分钟/每天重置)
- **DeepL Free**: 服务停止,下月1号恢复
- **DeepL Pro**: 自动计费,继续使用

### Q7: 翻译速度慢怎么办?
**A:**
- Gemini通常1-2秒
- DeepL通常2-3秒
- 如果很慢,检查:
  - 网络连接
  - API密钥是否正确
  - 是否达到速率限制

### Q8: 可以翻译其他语言吗?
**A:**
- 当前版本: 仅支持日文→中文
- Gemini支持: 100+语言
- DeepL支持: 30+语言
- 后续版本可能添加更多语言支持

### Q9: 翻译结果不满意怎么办?
**A:**
- 切换不同的API试试
- 手动编辑翻译结果
- 保存修改后的版本到记忆库
- 下次相似内容会优先显示你的版本

### Q10: 企业使用有什么建议?
**A:**
- 使用DeepL Pro获得最佳质量
- 设置专用API密钥(不共享个人密钥)
- 定期导出翻译数据备份
- 考虑设置使用配额限制
- 可以为不同团队配置不同API
